# AI Text Summarizer

Production-ready AI text summarizer built with Next.js App Router, TypeScript, Tailwind CSS, and a fully free open-source local model runtime.

## Features

- Summary modes: short, bullet points, detailed
- Input validation and sanitization
- Basic in-memory rate limiting
- Responsive, polished UI with loading and error states
- Copy-to-clipboard support
- No API keys required

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- @xenova/transformers (open-source local inference)

## Setup

1. Install dependencies:

```bash
npm install
```

2. No `.env.local` setup is required for default local inference.

3. Run development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## API Endpoint

`POST /api/summarize`

Request body:

```json
{
  "text": "Text to summarize",
  "summaryType": "short"
}
```

Valid `summaryType` values:

- `short`
- `bullets`
- `detailed`

The endpoint validates text, sanitizes input, rate limits requests, and returns a summary generated by a local open-source model.
